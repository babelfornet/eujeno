# P2P senza coordinator — nodi `serve` interrogabili (entry simmetrico) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`).

**Goal:** Rendere ogni nodo `serve` (modalità gossip P2P) un **entry interrogabile**: espone `/v1/chat/completions` (e `/v1/models`) orchestrando l'inferenza sulla **topologia gossipata** (`build_chain`) via **transport diretto HTTP**. Così il coordinator diventa opzionale e dal frontend si può creare/usare una rete **P2P pura**.

**Architecture:** Un motore di generazione condiviso `generate_tokens` (loop autoregressivo con chat template + sampling + stop EOS, **transport iniettato** via closure async). Il nodo `serve` (BlockServer) costruisce la catena dal proprio `Registry` gossipato e fornisce closure che chiamano via HTTP gli endpoint `/embed`,`/decode/{block}`,`/head` dei peer. Frontend: modalità **P2P** nel tab Gestione (`serve --peers/--advertise`). La UI punta la dashboard/chat a un peer qualsiasi (che ora ha `/registry` *e* `/v1`).

**Tech Stack:** Python · FastAPI/httpx · l'esistente `synapse/net/{server,discovery,sampling,tools,wire}.py`.

**Nota:** il coordinator (modalità relay/NAT) resta invariato. Niente failover nel serve-entry v1 (single pass). De-dup col loop del coordinator: follow-up.

    async def head(job_id: str, request: Request, topk: int = 1):

```python
import random

import torch

from synapse.net.sampling import sample_token


async def generate_tokens(tokenizer, prompt, max_new, sampling, stop_ids,
                          run_embed, run_decoders, run_head):
    """Loop autoregressivo con transport iniettato.
    run_embed(cur_ids)->hidden ; run_decoders(hidden, cache_position)->hidden (tutti i blocchi) ;
    run_head(hidden, topk)->{token_id, topk_ids, topk_logits}. Ritorna (tokens, prompt_len, finish_reason)."""
    temperature = float(sampling.get("temperature", 0.0) or 0.0)
    top_p = float(sampling.get("top_p", 1.0) or 1.0)
    rep = float(sampling.get("repetition_penalty", 1.0) or 1.0)
    do_sample = temperature > 0.0
    generator = None
    if do_sample:
        seed = sampling.get("seed")
        seed = int(seed) if seed is not None else random.randint(0, 2**31 - 1)
        generator = torch.Generator().manual_seed(seed)
    topk = 100 if do_sample else 1

    ids = tokenizer(prompt, return_tensors="pt").input_ids
    seq_len = ids.shape[1]
    cache_position = torch.arange(seq_len)
    cur = ids
    tokens = []
    finish_reason = "length"
    for step in range(max_new):
        h = await run_embed(cur)
        h = await run_decoders(h, cache_position)
        rh = await run_head(h, topk)
        if do_sample:
            tok = sample_token(rh["topk_ids"], rh["topk_logits"], tokens, temperature, top_p, rep, generator)
        else:
            tok = rh["token_id"]
        if tok in stop_ids:
            finish_reason = "stop"
            break
        tokens.append(tok)
        cur = torch.tensor([[tok]])
        cache_position = torch.tensor([seq_len + step])
    return tokens, seq_len, finish_reason
```
